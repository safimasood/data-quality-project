# Data Quality & Cleansing Project

## Overview
This project focuses on improving data quality by identifying and fixing issues such as duplicates, missing values, and inconsistencies in customer data.

## Tools Used
- MySQL
- Power BI

## Data Quality Issues Identified
- Duplicate records
- Missing values
- Invalid email formats
- Inconsistent text formatting

## Data Cleaning Techniques
- Standardization (UPPERCASE)
- Deduplication using SQL
- Null handling
- Email validation

## KPIs
- Completeness
- Uniqueness
- Validity

## Dashboard
Power BI dashboard showing:
- Data quality metrics
- Data distribution
- Issue tracking

## Outcome
Improved data accuracy and consistency for better decision-making.
