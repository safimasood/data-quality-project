# Data Quality & Cleansing Project

## Overview
This project focuses on improving data quality by identifying and fixing issues such as duplicates, missing values, and inconsistencies in customer data.

## Tools Used
- MySQL
- Power BI

## Data Issues
- Duplicate records
- Missing values
- Invalid emails
- Inconsistent formatting

## Data Cleaning
- Standardization
- Deduplication
- Validation rules

## KPIs
- Completeness
- Uniqueness
- Validity

## Dashboard
Power BI dashboard showing data quality metrics.
